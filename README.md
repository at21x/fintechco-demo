# fintechco-demo

> A FinTech demo designed to test Claude Code's capabilities — not a real company or system.

This repository contains a simulated payments validation microservice
(`payment_service/`) used for exploring code patterns and workflows. There is
no real web server, database, or payment processor involved — it's a
sandbox, not production software.

See [`payment_service/CLAUDE.md`](payment_service/CLAUDE.md) for details on
the module layout and conventions.
